1) Anatomy of a Prompt

Use this annotated template to explain how a high‑quality instruction is structured. You can edit the text live and copy it for your slides or LMS.

Editable Template

Where each piece fits

ROLE/CONTEXT → sets perspective & domain
TASK → the primary action
CONSTRAINTS → rules to follow / ignore
INPUT → the problem or data
OUTPUT FORMAT → shape of result
EVALUATION → self‑checks/double‑checks
ETHICS/BOUNDARY → when to refuse or pivot

Tip: Make constraints explicit. Adversarial content often tries to overwrite these.

Ethical use only. Use these attacks to teach robustness and academic integrity, and to design stronger defensive prompts.
